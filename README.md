# 👨‍💻 Kan Kılıç - Kişisel Portfolyo Web Sitesi

Bu repo, kişisel portfolyo web sitemin kaynak kodlarını içermektedir. Yazılım mühendisliği öğrencisi olarak geliştirdiğim projeleri, yeteneklerimi ve iletişim bilgilerimi bu sitede sergiliyorum.

🔗 **Canlı Site:** [Web Siteme Gitmek İçin Tıklayın](https://kanklc34.github.io/portfolyo/)

## 🚀 Projelerim

Sitede yer alan başlıca projelerim:

1.  **Personel Puantaj Sistemi (PWA):** JavaScript ve LocalStorage ile çalışan, Excel raporlama özellikli internetsiz takip sistemi.
2.  **Envanter Depo Yönetim Sistemi:** C# ve SQL tabanlı, chatbot destekli stok takip uygulaması.
3.  **Kişisel Portfolyo:** Şu an incelediğiniz responsive web sitesi.

## 🛠️ Kullanılan Teknolojiler

Bu web sitesi tasarlanırken aşağıdaki teknolojiler kullanılmıştır:

* **HTML5:** Semantik yapı ve içerik.
* **CSS3:** Flexbox ve Grid sistemleri ile Responsive (Mobil Uyumlu) tasarım.
* **FontAwesome:** İkon setleri.
