# Portfolyom

Modern, responsive ve performans odakli kisisel portfolyo sitesi.

## Canli Demo

`https://kankilic.pages.dev`

## Ozellikler

- Responsive tasarim (mobil/tablet/masaustu)
- Sistem temasini takip eden light/dark gorunum
- Guvenli dis linkler (`rel="noopener noreferrer"`)
- Optimize proje gorselleri (`.webp`)
- CV indirme butonu ve temel GA4 event takibi

## Teknoloji Stack

- HTML5
- CSS3
- Vanilla JavaScript

## Kurulum

1. Repoyu klonla: `git clone https://github.com/kanklc34/portfolyo.git`
2. Klasore gir: `cd portfolyo`
3. `index.html` dosyasini tarayicida ac veya Live Server ile calistir.

## Proje Yapisi

```text
.
|- index.html
|- css/
|  |- style.css
|- js/
|  |- main.js
|- images/
|  |- projects/
|- cv/
|  |- kan-kilic-cv.pdf
```
