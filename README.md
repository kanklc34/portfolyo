## 🚀 Yerelde Çalıştırma

```bash
# 1. Repoyu klonla
git clone https://github.com/kanklc34/portfolio.git
cd portfolio

# 2. Basit bir HTTP sunucusu başlat
# Python 3 ile:
python -m http.server 8000

# Veya Node.js ile:
npx serve .

# Veya VS Code Live Server eklentisi ile

# 3. Tarayıcıda aç
http://localhost:8000
